Inline Question 1:¶

It is possible that once in a while a dimension in the gradcheck will not match exactly. What could such a discrepancy be caused by? Is it a reason for concern? What is a simple example in one dimension where a gradient check could fail? Hint: the SVM loss function is not strictly speaking differentiable

Your Answer: fA discrepancy may happen when the loss function is not differentiable at some certain points. The function rectifier $f(x) = max(0,x)$ is not differetiable at x=0. The numerical gradient formula is $\frac{df(x)}{dx}= \frac {f(x+h)-f(x-h)}{2h} $. By this formula, the numerical gradients is $f'(\frac{h}{2}) = \frac{3}{4}$, while the analytic gradient is $f'(\frac{h}{2}) = 1$. Thus, a discrtpancy will be caused at the point around $x=0$.*
